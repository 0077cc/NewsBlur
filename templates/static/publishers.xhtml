Control the Original View

To control where stories begin in the Original view, you can use any of these class names in your HTML:

  • .entry
  • .post
  • .postProp
  • .article
  • #postContent

If NewsBlur cannot find one of these class names, it will check for the following, in order:

  • Exact match in a single visible element.
  • Near-match by header tags: h1, h2, h3, h4, h5, h6.
  • Near-match in a .post or similar class name.
  • Near-match by font-size (largest font size wins).
  • Each match is ranked by similarity.

To view the exact heuristics used, see find_story_in_feed_iframe in reader.js on GitHub.
